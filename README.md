# Verter Music API
Simple nodejs api to fetch information from different plattforms about verter music.

## Endpoints

Get all albums
```
/api/albums 
```

Get album by exact name
```
/album/:name
```

Get all tracks
```
/api/tracks
```

Get track by exact name
```
/api/track/:name
```

Get all albums alongside their tracks
```
/api/all
```
